✅ Version 2.21 - Added Clearing Account Entries

This version now includes two additional clearing entries for each invoice: a credit entry to the receivables account and a debit entry to the clearing account (default 1652, configurable).

🎯 Key Addition in v2.21: Two new journal entries are added for clearing purposes - offsetting the receivables account and posting to a configurable clearing account.

🆕 Changes in v2.21:
  • Clearing Entries - Added 4th entry (receivables credit) and 5th entry (clearing account debit)
  • Configurable Clearing Account - New setting for clearing account (default: 1652)
  • All v2.20 features retained: OSS structure fix, currency fields, VAT detection, etc.
